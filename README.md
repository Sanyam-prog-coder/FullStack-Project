# Full Stack Project

## Description
This is a Full Stack Web Application built using Spring Boot for backend and React for frontend. The application manages batch entries and demonstrates REST API integration.

## Technologies Used
- Java
- Spring Boot
- REST API
- MySQL
- ReactJS
- Node.js
- Maven

## Features
- Add Batch Entry
- View Batch Data
- Backend REST API
- React Frontend UI
- Database Integration

## Project Structure
FullStack-Project
│
├── MarvellousFullStack (Spring Boot Backend)
└── marvellous-frontend (React Frontend)
